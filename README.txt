
Lua Chatterbot
--------------

This is a highly unoptimized, buggy, slow, inefficient, and funny chat bot
using Markov chains.

Too lazy to document. Copy `chatterbot.conf.example' to `chatterbot.conf',
read the comments and modify.

Start with:

  $ luajit main.lua

or:

  $ chmod +x main.lua
  $ ./main.lua
